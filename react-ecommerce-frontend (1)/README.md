# React E-commerce Frontend (No Backend)

This is a simple React frontend-only e-commerce starter project.

Features:
- Product listing (static data)
- Add to cart / Remove / Clear
- Cart total & checkout placeholder
- Local React state (no backend or persistence)

How to run:
1. `npm install`
2. `npm run dev` (requires Vite)

This project is intentionally frontend-only for portfolio/demo use.
